# Threelayer_perceptron
2入力OR・AND回路を出力する3層パーセプトロン</br>

今回の目的</br>
0 0 =>0 0</br>
0 1 =>0 1</br>
1 0 =>0 1</br>
1 1 =>1 1</br>
になるように学習させる
